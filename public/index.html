Загрузка месяца из Excel (общая выручка)

Загрузите файл Excel с колонками «Операционный день», «Продажи», «Гостей», «Чеков». Если есть несколько касс, значения по дате будут автоматически сложены.

После загрузки появится страница с результатом и ссылкой «Назад».

Загрузка по официантам из Excel

Загрузите файл Excel, где по строкам указаны смены официантов. Ожидаемые колонки (названия могут немного отличаться, но будут распознаны по смыслу):

  • Дата смены: «Операционный день» или «Дата»
  • Официант: «Официант», «Сотрудник», «ФИО»
  • Выручка: «Выручка», «Продажи»
  • Гостей: «Гостей», «Гости»
  • Чеков: «Чеков», «Чеки»
  • Блюда (БКВ): «Блюда», «БКВ»

Для каждой даты и официанта показатели будут сложены. Перед записью старые данные официантов за эти даты будут удалены.
